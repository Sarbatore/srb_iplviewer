# Features
- Simply toggle ipls for a powerful dev workflow
- Complete collection of IPLs

# Installation
1. Drop `srb_iplviewer` in your `resources` folder
2. Add `ensure srb_iplviewer` in `server.cfg`

# Use
1. Type `ipl` command
2. Press `ALT` to enable mouse cursor
3. `LEFT CLICK` on texts to toggle IPL and print in console
